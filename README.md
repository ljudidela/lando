# Los Santos WebGL Landing

A GTA San Andreas inspired landing page featuring a 3D environment, procedural audio, and retro UI.

## Features
- Three.js 3D Background with procedural city generation
- Custom cursor and hover effects
- Procedural Audio Beat (Web Audio API)
- Retro GTA-style UI and Fonts
- Loading screen simulation

## How to Run

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open the link provided in the terminal (usually http://localhost:5173).
